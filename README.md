# TonyGL

Just a little library for rendering WebGPU named after my dog.

Currently supports:

- Scene graph
- Perspective camera
- Indexed and non indexed geometry support
- Geometry tangent generation
- Lambertian material
- Unlit material
- Normal material
- Alpha maps
- Albedo texture maps
- Multisampling
- Sorted alpha blending
- Point lights
- Scene level ambient light
- Texture resource handling
- Multipass rendering
- Linear -> sRGB workflow
- ACES Tonemapping

Full documentation to come.
